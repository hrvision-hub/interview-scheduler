# Страница записи на собеседование

Кликабельный прототип страницы, где кандидат выбирает удобный день и время интервью. Данные кандидата вручную не вводятся: предполагается, что в боевой версии они будут передаваться из CRM или другого источника.

## Файлы

- `interview-scheduler.html` — разметка страницы.
- `interview-scheduler.css` — визуальный стиль, адаптивность, карточки, кнопки и состояния.
- `interview-scheduler.js` — логика выбора даты, слота и подтверждения записи.

## Как открыть

Откройте `interview-scheduler.html` в браузере или запустите локальный сервер из папки с файлами.

Пример локального адреса в текущей папке:

```text
http://127.0.0.1:4173/interview-scheduler.html
```

## Что уже есть

- Автоматическое определение сегодняшней даты.
- Показ только текущего и следующих доступных дней.
- Выбор дня и времени интервью.
- Подтверждение записи без ручного ввода ФИО, почты или Telegram.
- Информационные карточки: длительность, формат, напоминание, ссылка на встречу.
- Адаптивная верстка для мобильных экранов.

## Что заменить при интеграции

В `interview-scheduler.js` сейчас используются демо-слоты. Для реального сценария нужно заменить функцию `getSlotsForDate()` на получение свободных слотов из календаря рекрутера или backend/API.

Имя кандидата сейчас задано в переменной:

```js
const candidateName = "Анастасия";
```

В боевой версии его можно подставлять из CRM, ссылки-приглашения или параметров страницы.

## Дальнейшие улучшения

- Сохранение выбранного слота в CRM.
- Блокировка уже занятых слотов.
- Отправка автоматического напоминания кандидату.
- Передача рекрутеру подтверждения записи.
- Генерация или отправка ссылки на видеовстречу.
